# Story Land

Welcome to **Story Land**, the playful and interactive app designed for children to create and enjoy personalized stories!

## Features

- **Personalized Stories**: Generate stories tailored to the child's age, name, and favorite keywords.
- **Categories**: Choose from various categories including Animals, Bedtime, Bible Stories, Plants, African Folklore, and African History.
- **Interactive Reading**: Text-to-Speech functionality to read stories aloud.
- **Share Stories**: Share your generated stories with friends and family.
- **Playful Design**: A colorful and engaging interface that appeals to children.

## Screens

### 1. Welcome Screen
- Choose the age of the child.
- Proceed to select a story category.

### 2. Story Input Screen
- Enter the child's name.
- Select a story category.
- Enter a keyword related to the story.
- Generate a personalized story.

### 3. Story Display Screen
- View the generated story.
- Use Text-to-Speech to read the story aloud.
- Share the story with others.
- Option to generate another story.

## Technologies Used

- **Flutter**: For building the cross-platform application.
- **Provider**: For state management.
- **Google Generative AI**: For generating stories.
- **Flutter TTS**: For Text-to-Speech functionality.
- **Flutter Dotenv**: For managing environment variables.

## Installation

1. **Clone the repository:**
    ```sh
    git clone https://github.com/Atinuke-Manuels/StoryLand.git
    cd story-land
    ```

2. **Install dependencies:**
    ```sh
    flutter pub get
    ```

3. **Set up environment variables:**
    - Create a `.env` file in the root directory.
    - Add your API key:
      ```env
      API_KEY=your_api_key_here
      ```

4. **Run the app:**
    ```sh
    flutter run
    ```
